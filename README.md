# Usage
You may use this repository's contents to build a site of your own.  

1. Clone the repo. Make sure to have Git installed.
```
apt install git
```
(or any package manager you use.)
```
git clone https://bluezcreen.github.io
```

2. Edit the contents as you want in a text editor/IDE.
3. To test the site before publishing, you may want to use localhost.
```
apt install python
cd path/to/file.html
python3 -m http.server 8000
```
